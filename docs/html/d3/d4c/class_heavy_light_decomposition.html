Detailed Description

template<typename E>
class HeavyLightDecomposition< E >

重軽分解

各パスを色で塗っていく col[v] = col[u] のときは同じ成分 col のナンバリング順は DFS ordering とする。ある成分の子にあたる成分は必ず番号が大きくなる
